# Gleditsia home photo placeholders

The SVG files in this folder are local placeholder images for the First Part prototype. They are intentionally quiet, gray or muted, and only exist to test the responsive mosaic layout.

When real work is ready, replace these files with photographs or update `src/data/homePhotos.ts` to point to the final image filenames. Do not use `UI_design/` reference images as final homepage photo content.
